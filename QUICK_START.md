# FocusTube - Quick Start Guide

## 🎯 Main Feature: Type Keywords to Block Videos

This is the core feature you asked for. Here's how it works:

### How to Use
1. **Load extension** → Go to `chrome://extensions` → Click "Load unpacked" → Select `/dist` folder
2. **Open settings** → Right-click FocusTube icon → Click "Options"
3. **Add keywords** → Type a keyword (e.g., "reaction") in the input field under "Blocked keywords"
4. **Click Block** → The keyword is now active
5. **Go to YouTube** → Videos with that keyword will be HIDDEN automatically

### Example Keywords You Can Add
- `movie` - Blocks all movies and films
- `reaction` - Blocks reaction videos
- `trending` - Blocks trending content
- `viral` - Blocks viral videos
- `comedy` - Blocks comedy content
- `shorts` - Blocks YouTube Shorts
- `music` - Blocks music videos
- `entertainment` - Blocks entertainment content

### What Happens
✅ When you type a keyword like "reaction":
- All videos with "reaction" in the title get HIDDEN
- All videos with "reaction" in the description get HIDDEN
- All videos from channels with "reaction" in the name get HIDDEN
- A status panel shows in top-right: "X videos blocked on this page"

### Visual Feedback
📍 **Top-right corner status**: Shows total blocked videos and why (e.g., "1 Custom keyword block")
📍 **Keywords list**: Each keyword appears in a highlighted box with a remove button
📍 **Color coding**: Purple highlight indicates active blocking

### Manage Your Keywords
- **Remove**: Click the X button on any keyword
- **View**: All active keywords listed under "Blocked keywords" section
- **Tips**: Shows "Your Custom Keywords" section with helpful info

### Quick Add Common Ones
If you have no keywords, click suggested keywords to quick-add:
- movie

## 🎮 Other Cool Features (Optional)

### Study Mode
- Automatically hides ALL entertainment content
- Only allows educational videos (coding, tutorials, etc.)
- Toggle: Settings → "Enable study mode"

### Block Entire Channels
- Click "Block this channel" button on any YouTube channel
- That entire channel's videos will be hidden

### Productivity Dashboard
- When you visit YouTube home, see your daily goal instead of feed
- Motivational quote and productivity timer
- Toggle: Settings → "Show productivity homepage"

### Strict Mode
- Set a password to prevent disabling the extension
- Add a delay (e.g., 30 seconds) before you can disable
- Useful when you want real commitment

### Check Your Stats
- See how many videos you've blocked
- Track your focus streak (consecutive days with blocks)
- Category breakdown of what you blocked
